# Sree Gonesh Info - AI Automation Sales Funnel

This is a static, high-converting HTML/JS landing page designed for selling Custom AI Automation, Web Apps, and Chatbots. 

 Features
- **Zero Hosting Cost:** Runs completely on GitHub Pages.
- **WhatsApp Form Redirect:** Uses JS to send form details directly to your business WhatsApp (No backend PHP needed).
- **Razorpay Integrated:** Client-side Razorpay checkout setup.
- **Ultra-Premium UI:** Dark mode SaaS design with Bento Grids and Apple-style aesthetic.
